# rg role

Installs [ripgrep](https://github.com/BurntSushi/ripgrep), the `rg` recursive
search tool.

| Platform        | Install                |
| --------------- | ---------------------- |
| macOS           | Homebrew (`ripgrep`)   |
| Debian / Ubuntu | `apt install ripgrep`  |
| RHEL family     | `dnf install ripgrep`  |

Homebrew is ensured by the `homebrew` role dependency.
