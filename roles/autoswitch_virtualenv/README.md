# autoswitch_virtualenv role

Installs [zsh-autoswitch-virtualenv](https://github.com/MichaelAquilina/zsh-autoswitch-virtualenv)
into the oh-my-zsh custom plugins dir (as `autoswitch_virtualenv`) and enables it
via `plugins.d/25-autoswitch_virtualenv.zsh`. Depends on `zsh_config`.

> Automatically activates/creates a virtualenv when you `cd` into a project with
> a `.venv`/`requirements.txt`.
